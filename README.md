# JunkBox_Win_Lib
C++ Static Library for MS Windows

## DownLoad
```
git clone https://github.com/JunkBox-Library/JunkBox_Win_Lib.git
```

## JunkBox_Lib++ is required
```
git clone https://github.com/JunkBox-Library/JunkBox_Libpp.git JunkBox_Lib++
```

### DX9 SDK
